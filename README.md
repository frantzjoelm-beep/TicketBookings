# TicketBooking Flask App

Application web de réservation de tickets développée avec Flask.

## Installation et exécution

### 1. Cloner le projet

```bash
git clone https://github.com/ton-utilisateur/TicketBooking.git
cd TicketBooking
```

### 2. Créer et activer un environnement virtuel

Sous Windows :

```bash
python -m venv venv
venv\Scripts\activate
```

Sous Linux/macOS :

```bash
python3 -m venv venv
source venv/bin/activate
```

### 3. Installer les dépendances

```bash
pip install -r requirements.txt
```

### 4. Lancer l'application

```bash
python run.py
```

### 5. Ouvrir l'application

Dans le navigateur, ouvrir :

```text
http://127.0.0.1:5000
```

## Technologies utilisées

- Python
- Flask
- Flask-Login
- SQLAlchemy
- SQLite
